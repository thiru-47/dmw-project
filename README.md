# 🎓 Student Performance Prediction System

A Streamlit-based Data Mining project that predicts student marks using Machine Learning.

## 🚀 Features
- Data preprocessing & visualization
- Linear Regression model
- Clustering (K-Means)
- OLAP-style analysis
- Prediction with optional saving
- Interactive dashboard

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## 📊 Dataset
Student_Marks.csv

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run dmw.py
